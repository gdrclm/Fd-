# Бургер-калькулятор

Мини-игра на HTML/CSS/JS: вместо цифр используются ингредиенты бургера.
Уровень засчитывается, если собрать ингредиенты в правильной последовательности.

## Локальный запуск

Откройте `index.html` в браузере.

## Мобильная версия

Интерфейс адаптивный:
- на узких экранах кнопки растягиваются на ширину контейнера;
- уменьшаются отступы и шрифты;
- блок управления перестраивается в колонку.

## Деплой на GitHub Pages

В репозитории добавлен workflow `.github/workflows/deploy-pages.yml`.

Что нужно сделать один раз в GitHub:
1. `Settings` → `Pages` → `Source: GitHub Actions`.
2. Убедиться, что основная ветка называется `main` (или обновить ветку в workflow).
3. Сделать push в `main` — workflow автоматически задеплоит сайт.

После успешного запуска Actions сайт будет доступен по адресу:
`https://<ваш-логин>.github.io/<имя-репозитория>/`
